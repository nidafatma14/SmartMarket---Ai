# 🚀 SmartMarket AI - Stock Sentiment Analysis
A Final Year Project that combines **Machine Learning**, **NLP**, and **Technical Analysis** to predict stock movements.

## ✨ Features
- **AI Scanner:** Ranks NIFTY stocks by bullish probability.
- **FinBERT Sentiment:** Analyzes news from Moneycontrol & Reuters.
- **Technical Indicators:** RSI, MACD, and Moving Averages.
- **React Dashboard:** Professional fintech UI with dark mode.

## 🛠️ Tech Stack
- **Frontend:** React + Tailwind CSS + Lucide Icons
- **Backend:** Python + FastAPI + Scikit-Learn
- **NLP:** FinBERT (HuggingFace Transformers)
- **Data:** Yahoo Finance API

## 🏃 How to Run
1. **Backend:** `python main.py`
2. **Frontend:** `npm install` then `npm run dev`
